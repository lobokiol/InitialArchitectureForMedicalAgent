# 前端面试演示页面设计提示词

> 基于 COSTAR 框架编写的 AI Agent 提示词，用于生成医院导诊系统的前端面试演示页面
>
> **更新说明**：整合了原 CLI 设计提示词中的核心功能（用户管理、会话管理、命令系统等）

---

## C (Context) 背景

你是一名资深前端开发工程师，擅长使用 React、TypeScript、Tailwind CSS 和 Framer Motion 构建现代化、交互性强的 Web 应用。你的任务是为一个**医院导诊 Agentic 助手**项目设计并实现一个用于面试演示的展示页面。

该项目是一个生产级的医院导诊问答系统，技术栈为：
- 后端：FastAPI + LangGraph + Redis + Elasticsearch + Milvus + DashScope
- 核心功能：症状问诊、多轮对话、RAG（检索增强生成）、意图识别
- 当前状态：后端 API 已开发完成，需要一个可视化的 Web 演示页面

---

## O (Objective) 目标

请根据上述项目背景，使用 **React + Tailwind CSS + Framer Motion** 构建一个用于面试演示的前端页面，该页面需要：

1. **用户管理**：支持输入 user_id 和昵称，可创建/获取用户信息
2. **会话管理**：支持创建会话、切换会话、删除会话、查看会话列表
3. **命令系统**：支持斜杠命令（/help、/new、/threads、/switch、/delete、/user）
4. **实时对话**：能够连接后端 `/chat` API 实现真实的对话演示
5. **意图展示**：显示意图识别结果（症状问诊、流程咨询、混合意图等）
6. **文档溯源**：显示本轮参考的文档来源（medical / process）
7. **视觉吸引**：使用 Framer Motion 实现流畅的动画效果
8. **面试友好**：结构清晰、重点突出，便于面试官快速了解项目能力

## 核心功能详细说明

### 1. 用户系统

- **首次访问**：弹出用户设置弹窗，要求输入 user_id（可默认 `demo-user`）和昵称（可选）
- **保存状态**：使用 localStorage 缓存用户信息
- **用户切换**：通过 `/user` 命令可修改用户信息

### 2. 会话管理

- **当前会话**：顶部显示当前用户和会话信息
- **会话列表**：通过 `/threads` 查看所有会话
- **创建会话**：通过 `/new` 创建新会话
- **切换会话**：通过 `/switch` 切换到其他会话
- **删除会话**：通过 `/delete` 删除会话

### 3. 对话功能

- **消息发送**：输入框发送消息，支持 Enter 发送
- **加载状态**：发送时显示"正在思考..."的加载动画
- **Markdown 渲染**：助手回复支持 Markdown 格式
- **意图识别**：显示识别到的意图类型（症状/流程/混合/非医疗）
- **文档溯源**：显示本轮参考的文档列表及其相似度

### 4. 错误处理

- **网络错误**：显示"网络错误，请检查后端服务"提示
- **API 错误**：显示具体错误信息
- **超时处理**：60秒超时提示
